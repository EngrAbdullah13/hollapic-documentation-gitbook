# Hollapic overview

Hollapic is an AI ecommerce image generation application for creating product images, lifestyle scenes, banners, mockups, labels, and collection visuals from uploaded product images.

## What Hollapic does

Hollapic combines product uploads, workflow-specific instructions, optional reference images, Brand DNA, and a prompt-planning model to create a final image prompt for the image generation model.

## Core capabilities

| Area | Capability |
| --- | --- |
| Image generation | Multiple ecommerce workflows, aspect ratios, quality settings, output download, and generation history. |
| Product protection | Prompt planning emphasizes preserving product shape, labels, logos, packaging colors, proportions, and visible identity. |
| Brand DNA | Stores reusable brand context and applies it as private visual guidance during generation. |
| Credits and billing | Uses subscription, purchased, and bonus credit buckets with Stripe checkout and webhooks. |
| Shared access | Owners can invite collaborators to generate inside the owner workspace with controlled credit access. |
| Admin tools | Internal prompt testing, app image management, and team credit controls. |
