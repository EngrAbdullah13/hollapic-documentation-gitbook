# Quick start

Use this path to generate a first ecommerce-ready image quickly.

## Before you begin

- Sign in to Hollapic.
- Have at least one usable credit.
- Prepare a clear product image.

## Steps

1. Open the Studio.
2. Upload the product image into the product image node.
3. Select a generation workflow.
4. Choose an aspect ratio and quality.
5. Add optional instructions if needed.
6. Select **Generate**.
7. Review the output.
8. Download the image or reuse the job from history.

## Expected result

Hollapic queues the job, plans the prompt, generates an output image, saves it to history, and makes the output available for download.
