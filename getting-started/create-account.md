# Create an account

Hollapic uses email-based OTP verification for signup and login.

## Before you begin

- Use an email address you can access.
- Keep the browser open while you wait for the code.

## Steps

1. Open Hollapic.
2. Choose the signup or login action.
3. Enter your email address and any requested profile fields.
4. Select **Send code**.
5. Check your inbox for a six-digit code.
6. Enter the code and continue.

## Expected result

After verification, Hollapic creates or restores your account and opens the authenticated product experience.

## Important behavior

- New users receive signup credits when eligible.
- OTP send attempts are rate limited per email address.
- Verification is separate from sending; rate limiting blocks new codes, not valid code verification.
