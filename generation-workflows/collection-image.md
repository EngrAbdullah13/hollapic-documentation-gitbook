# Collection Image

Collection Image creates a scene showing multiple products together.

## Before you begin

- Upload multiple product images when you want a grouped arrangement.
- Add an optional reference/style scene when available.

## Current image order behavior

For Collection Image, the prompt planner treats the first image as Product 1, the second image as Product 2, the third image as optional Product 3, and the fourth image as an optional reference/style scene.

## Expected result

The output should arrange the products as a polished ecommerce collection while preserving the main products.
