# Background Replace

Background Replace preserves the uploaded product composition and changes only the background or environment.

## Before you begin

- Upload the product image.
- Upload a reference image when the UI requires one for the workflow.

## Steps

1. Select **Background Replace**.
2. Upload the product image first.
3. Add the reference environment image if required.
4. Add optional instructions for lighting, blur, camera angle, or mood.
5. Generate the image.

## Expected result

The product remains the source of truth while the background/environment changes to match the requested direction.

## Important behavior

The planner instructs the model to preserve main products and product-related people when they are meaningfully connected to the product presentation.
