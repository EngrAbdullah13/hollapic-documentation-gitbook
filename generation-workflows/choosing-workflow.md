# Choose the correct workflow

Choose the workflow based on what should change.

## Decision table

| If you want to... | Use |
| --- | --- |
| Build a full commercial image around a product | [General Image Generation](general-image-generation.md) |
| Keep the product but change the environment | [Background Replace](background-replace.md) |
| Put your product into another scene | [Product Swap](product-swap.md) |
| Match lighting, mood, composition, and visual direction | [Style Copy](style-copy.md) |
| Create label or packaging design concepts | [Label Generation](label-generation.md) |
| Create marketing hero/banner creative | [Image Banner](image-banner.md) |
| Show packaging in realistic mockup form | [Packaging Mockup](packaging-mockup.md) |
| Show multiple products together | [Collection Image](collection-image.md) |
| Add structured feature callouts | [Product Infographic](product-infographic.md) |

## Common mistake

Do not use Product Swap when you only want a new background. Product Swap replaces the main product in the reference scene with your uploaded product.
