# Label Generation

Label Generation creates professional packaging or label concepts.

## Before you begin

- Upload a logo image when designing around a brand mark.
- Add a product or container image if the label must fit a specific surface.
- Add a reference design only as style inspiration.

## Steps

1. Select **Label Gen**.
2. Upload the logo or brand source image.
3. Add optional product/container and reference images.
4. Provide label direction or required words if any.
5. Generate the design.

## Expected result

The output should be a realistic packaging or label concept, guided by the uploaded brand source and optional product/reference images.

## Important behavior

Only include new visible words when the user explicitly supplied those exact words for the current generation.
