# Reference-image behavior

Reference images provide visual direction, workflow context, or scene structure depending on the selected workflow.

## Workflow-specific behavior

| Workflow | Reference behavior |
| --- | --- |
| Background Replace | Reference guides background/environment, lighting, composition, scene style, and mood. |
| Product Swap | Reference supplies the scene where the uploaded product is placed. |
| Style Copy | Reference supplies visual style, lighting, mood, color grading, and composition. |
| Label Generation | Optional reference is style inspiration only. |
| Collection Image | Optional fourth image can be a reference/style scene. |

## Important behavior

Reference images should not cause competitor logos, slogans, package designs, brand names, exact people, or one-to-one artwork to be copied.
