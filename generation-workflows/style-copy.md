# Style Copy

Style Copy follows the visual style of a reference image while preserving your uploaded product.

## Before you begin

- Image 1 is the product source.
- Image 2 is the reference style image.

## Steps

1. Select **Style Copy**.
2. Upload the product image.
3. Upload the style reference image.
4. Add optional scene or mood instructions.
5. Generate the image.

## Expected result

The image should feel close to the reference in lighting, mood, color, composition, and photography style while remaining a new image and keeping your product identity.

## Important behavior

If a person appears in the reference, Hollapic’s planner instructs generation to create a different person in a similar role. It should not recreate or identify the same person.
