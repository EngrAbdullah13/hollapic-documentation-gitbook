# Product to Full Image

Product to Full Image is handled as a legacy alias for General Image behavior in the current backend.

## Current behavior

Requests using `product_to_full_image` normalize to the General Image workflow. The expected output is a complete commercial product or lifestyle scene built around the uploaded product.

## When to use it

Use General Image in the current Studio UI when you want the product expanded into a finished ecommerce scene.
