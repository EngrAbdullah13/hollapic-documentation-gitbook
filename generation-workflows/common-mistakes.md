# Common mistakes

Avoid these common setup issues.

## Using the wrong workflow

Use Background Replace when the product should stay in place and only the environment should change. Use Product Swap when the uploaded product should be inserted into a reference scene.

## Uploading low-quality product images

If labels, logos, or product edges are unclear in the source image, generation can be less reliable.

## Expecting exact text without providing it

Only request visible text when you provide the exact words required.

## Copying references too literally

Reference images should guide style, not copy another brand’s product, person, logo, slogan, or exact campaign artwork.
