# Manage collaborators

Owners can view pending invitations, active collaborators, access type, and credit usage from the Invite Team page.

## Actions

- Send a new invitation.
- Refresh team access data.
- Edit access type or limited credit allocation.
- Remove collaborator access.

## Important behavior

Every mutation returns updated collaborator data and a server-calculated credit summary. The UI should use that response as the source of truth.
