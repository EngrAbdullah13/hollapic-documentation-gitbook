# Accept an invitation

Collaborators accept workspace invitations using the invited email address.

## Before you begin

- Use the email address that received the invitation.
- Accept before the invitation expires.

## Steps

1. Open the invitation link.
2. Sign in or sign up with the invited email.
3. Confirm the invitation.
4. Switch to the shared workspace if it is not already active.

## Expected result

The shared workspace appears in the workspace switcher and the collaborator can generate using the access granted by the owner.

## Important behavior

Invitations expire after the configured invitation TTL. In the current backend, workspace invitation links are created with a seven-day TTL.
