# Remove access

Owners can revoke collaborator access.

## Steps

1. Open **Invite Team**.
2. Find the collaborator.
3. Select **Remove Access**.
4. Confirm the action if prompted.

## Expected result

The collaborator is marked inactive, cannot generate in the owner workspace, and no longer appears as active access. If the collaborator had limited access, unused reserved credits become free/shared again because inactive members are excluded from reservations.
