# Shared Workspace overview

Shared Workspace access lets an owner invite an agency or collaborator to work inside the owner’s Hollapic workspace without sharing passwords.

{% hint style="info" %}
Screenshot required: Workspace switcher in the generation-page header showing Personal Workspace and shared workspaces.
{% endhint %}

## Core rules

- Credits, billing, Brand DNA, history, workflows, and saved generation data belong to the workspace owner.
- Collaborators can generate and download inside the owner workspace when access is active.
- Collaborator generations are saved in the owner’s history.
- Collaborators cannot manage billing, invite others, or transfer credits in phase one.
