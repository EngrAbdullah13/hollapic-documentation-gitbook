# Generation history

Hollapic saves generation jobs and assets so users can review previous outputs.

## What history includes

- Job status.
- Input and output image references.
- Generation type.
- Prompt planning metadata used internally.
- Workspace owner and generated-by attribution for shared workspaces.

## Shared workspaces

Owner workspace history belongs to the owner. Collaborator generations are saved in the owner’s normal history and include generated-by attribution.
