# Select a generation workflow

Each workflow changes how Hollapic interprets uploaded images and instructions.

## Steps

1. Review the workflow list in Studio.
2. Pick the task that matches your desired output.
3. Upload the required images for that workflow.
4. Add optional answers or custom instructions.

## Workflow selection guide

| Goal | Recommended workflow |
| --- | --- |
| Create a full ecommerce/lifestyle image | General Image |
| Change only the scene behind a product | Background Replace |
| Place your product into a reference scene | Product Swap |
| Match the look of a reference image | Style Copy |
| Create a packaging or label concept | Label Generation |
| Create a marketing banner | Image Banner |
| Show packaging on realistic formats | Packaging Mockup |
| Show multiple products together | Collection Image |
| Create product benefit callouts | Product Infographic |
