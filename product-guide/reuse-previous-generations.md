# Reuse previous generations

Use history to inspect previous outputs and continue work from a completed generation.

## Steps

1. Open generation history.
2. Select a previous job.
3. Review the output and related settings.
4. Use the same workflow and inputs as a starting point for the next generation.

## Important behavior

Brand DNA summaries are snapshotted on generation jobs when applied. Editing Brand DNA later does not retroactively change already-created job snapshots.
