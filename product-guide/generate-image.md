# Generate an image

Generation creates a job, deducts credits, plans the prompt, calls the image model, stores the output, and updates history.

## Before you begin

- Upload required images.
- Confirm the active workspace has enough credits.
- Confirm collaborator access if you are working inside another owner workspace.

## Steps

1. Select the workflow.
2. Upload required images.
3. Choose aspect ratio and quality.
4. Add optional answers or custom instructions.
5. Confirm Brand DNA toggle if available.
6. Select **Generate**.

## Expected result

The job moves through pending, planning, generating, processing, and completed states. If generation fails, an error is stored and shown.
