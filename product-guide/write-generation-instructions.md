# Write generation instructions

Optional instructions help Hollapic understand the scene, audience, mood, lighting, props, and creative direction.

## Best practices

- Describe the desired commercial outcome.
- Mention specific scene details only when they matter.
- Use exact words only when you want visible text included.
- Do not ask the model to redesign the product if product identity must remain stable.

## Example

```text
Create a premium bathroom shelf lifestyle scene with soft daylight, marble surface, white towels, and fresh green accents. Keep the product label clear.
```

## Important behavior

The prompt planner uses user instructions and optional answers while preserving the uploaded product identity and workflow rules.
