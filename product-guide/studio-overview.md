# Hollapic Studio overview

Studio is the main workspace for uploading product images, choosing workflows, entering optional instructions, setting aspect ratio and quality, generating images, and reviewing outputs.

Screenshot required: Full Studio interface showing workflow cards, product upload node, aspect ratio and quality controls, output image node, optional questions, credits, workspace switcher, and user menu.
{% endhint %}

## Main areas

| Area | Purpose |
| --- | --- |
| Workflow selector | Chooses the generation behavior. |
| Upload nodes | Collect product and reference images required by the selected workflow. |
| Optional questions | Capture scene, audience, mood, lighting, props, and other guidance when available. |
| Aspect ratio and quality controls | Set output composition and model quality behavior. |
| Output node | Shows generated result and output actions. |
| Header | Shows credits, active workspace, Brand DNA access, and user menu. |
