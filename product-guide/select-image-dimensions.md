# Select image dimensions

Studio supports confirmed aspect ratios from the backend generation configuration.

## Supported aspect ratios

| Aspect ratio | Use case | Image model size |
| --- | --- | --- |
| 1:1 | Square ecommerce composition | 1024x1024 |
| 4:5 | Mobile-first portrait product images | 1024x1536 |
| 16:9 | Wide banners and horizontal creative | 1536x1024 |
| 4:3 | Balanced standard product composition | 1536x1024 |

## Steps

1. Open the aspect ratio dropdown above the output node.
2. Choose the ratio that matches your channel.
3. Generate or regenerate the image.
