# Download images

Completed outputs can be downloaded from the output node or history.

## Steps

1. Generate an image.
2. Wait until the output appears.
3. Select the download action in the output image node.
4. Save the file locally.

## Expected result

The browser downloads the generated image with a Hollapic job-based filename.

## Troubleshooting

If download fails, verify the job completed and the output image is visible. See [Download problems](../troubleshooting/download-problems.md).
