# Skip Brand DNA

Brand DNA onboarding can be skipped.

## Behavior

If a user skips Brand DNA, the saved status prevents onboarding from repeatedly appearing. The profile is not treated as completed and the generation toggle is not enabled from a completed profile.

## Complete it later

Open **Settings -> Brand DNA** to fill the profile later.
