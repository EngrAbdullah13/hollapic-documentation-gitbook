# Style references

Style references are currently handled by generation workflows, not by the saved Brand DNA profile.

## Best workflows for style references

| Need | Workflow |
| --- | --- |
| Match overall look and mood | Style Copy |
| Use a reference as background direction | Background Replace |
| Put product into an existing-style scene | Product Swap |

## Important behavior

Reference images influence visual language only. They should not copy competitor branding, product designs, slogans, logos, or exact artwork.
