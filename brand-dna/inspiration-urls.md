# Inspiration URLs

Inspiration URL fields are not part of the current Brand DNA backend schema.

## Current product behavior

The current Brand DNA profile stores one `websiteUrl` for the brand website. It does not store competitor or inspiration URL arrays.

## How to use inspiration today

Use workflow reference images instead:

- Use [Style Copy](../generation-workflows/style-copy.md) for reference-style direction.
- Use [Background Replace](../generation-workflows/background-replace.md) for environment direction.
- Use [Product Swap](../generation-workflows/product-swap.md) to place your product into a reference scene.

{% hint style="warning" %}
Do not use inspiration sources to copy another brand’s logo, product design, exact campaign, slogans, packaging, or person.
{% endhint %}
