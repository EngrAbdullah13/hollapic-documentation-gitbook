# Edit Brand DNA

You can update Brand DNA after onboarding.

## Steps

1. Open **Settings -> Brand DNA**.
2. Update brand name, products, audience, feel, colors, or website URL.
3. Save the profile.
4. Return to Studio and use the Brand DNA toggle during generation.

## Expected result

Future generations use the updated summary. Existing generation jobs keep their original saved Brand DNA snapshot.
