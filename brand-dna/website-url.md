# Website URL

The Website URL field stores the brand’s own public website URL.

## Validation

The backend accepts only public `http:` and `https:` URLs, requires a valid hostname, removes fragments and query strings, and rejects localhost or private-network addresses.

## Current behavior

The website URL is stored for the Brand DNA profile. It is not included directly in the generated brand summary.
