## Quick selection guide

| What you want to create | Generation type |
| --- | --- |
| A complete commercial or lifestyle image | **General Image Generation** |
| A new environment around the same product | **Background Replace** |
| Your product placed into another scene | **Product Swap** |
| A new image inspired by another image’s style | **Style Copy** |
| A product label or label concept | **Label Generation** |
| A wide website or advertising visual | **Image Banner** |
| A realistic packaging presentation | **Packaging Mockup** |
| Several products shown together | **Collection Image** |
| A product image with benefits or feature callouts | **Product Infographic** |
| A complete image created from a product source | **Product to Full Image** |

## Required and optional images

After you select a generation type, Hollapic shows the upload areas for that workflow.

- Upload every image marked as required.
- Optional images can be left empty.
- When image order matters, follow the labels shown in Studio.
