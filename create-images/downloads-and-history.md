# Downloads and History

You can download a finished image from Studio or return to it later through your generation history.

## Download an image

1. Wait until the generation is complete.
2. Review the result.
3. Select the **Download** action on the output image.
4. Choose where to save the file on your computer.

{% endhint %}

## Open generation history

Your history keeps previous generation jobs and outputs.

1. Open **Generation History**.
2. Select a previous generation.
3. Review the output and workflow.
4. Download the image or use the same idea as a starting point for another generation.

{% hint style="info" %}
Screenshot needed: Generation History showing several completed images, dates or statuses, and one selected item. Place this after the history steps.
{% endhint %}

## Shared workspaces

Images created by a collaborator inside an owner’s workspace appear in that workspace’s history.

## If a download does not work

- Confirm the image finished generating.
- Refresh the page and open the result again.
- Try downloading from history.
- See [Common Image Problems](../help/common-image-problems.md).
