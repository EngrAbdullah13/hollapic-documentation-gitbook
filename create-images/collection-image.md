# Collection Image

Use Collection Image to show several products together in one coordinated image.

<figure><img src="../.gitbook/assets/Screenshot 2026-07-16 174243.png" alt="" width="563"><figcaption></figcaption></figure>

## How to generate an image

1. Sign up or log in to Hollapic and open **Studio**.
2. Set up Brand DNA if you want Hollapic to follow your saved brand direction. This is optional.
3. Select **Collection Image**.
4. Upload the required product images in the order shown in Studio.
5. Add an optional reference image or instructions for arrangement, spacing, background, and visual style.
6. Choose the image size and resolution or quality.
7. Turn on Brand DNA if you have already set it up and want to use it for this generation.
8. Select **Generate**.

## Example instruction

> Arrange the products as a balanced skincare collection on a light stone surface with soft natural shadows.

## What happens next

Hollapic creates one collection image that presents the uploaded products together.

Review the product, label, logo, colors, text, and overall image before downloading or copying it.
