# Label Generation

Use Label Generation to create a product label or packaging-label concept.

## How to generate an image

1. Sign up or log in to Hollapic and open **Studio**.
2. Set up Brand DNA if you want Hollapic to follow your saved brand direction. This is optional.
3. Select **Label Generation**.
4. Upload the required logo, product, container, or brand source images shown in Studio.
5. Add the exact text, label direction, colors, and layout details as optional instructions when needed.
6. Choose the image size and resolution or quality.
7. Turn on Brand DNA if you have already set it up and want to use it for this generation.
8. Select **Generate**.

## Example instruction

> Create a clean premium skincare label using the uploaded logo and brand colors. Keep the product name clear and easy to read.

## What happens next

Hollapic creates a label concept based on the images and details you provide.

Review the product, label, logo, colors, text, and overall image before downloading or copying it.
