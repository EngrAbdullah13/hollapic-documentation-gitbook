# Image Banner

Use Image Banner to create a wide ecommerce, website, campaign, or advertising image.

## How to generate an image

1. Sign up or log in to Hollapic and open **Studio**.
2. Set up Brand DNA if you want Hollapic to follow your saved brand direction. This is optional.
3. Select **Image Banner**.
4. Upload the required product images shown in the banner workflow.
5. Add optional instructions for headline space, product position, background, mood, or campaign style.
6. Choose the image size and resolution or quality.
7. Turn on Brand DNA if you have already set it up and want to use it for this generation.
8. Select **Generate**.

## Example instruction

> Create a wide website hero banner with the product on the right and clean space on the left for a headline.

## What happens next

Hollapic creates a wide image designed for banner-style placement.

Review the product, label, logo, colors, text, and overall image before downloading or copying it.
