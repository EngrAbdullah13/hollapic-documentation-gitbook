# Common Image Problems

Use this page when a generated image does not look right.

## The product appearance changed

### Try this

1. Upload a clearer product image.
2. Add: “Keep the exact product, label, logo, colors, material, and packaging shape.”
3. Remove instructions that ask for a product redesign.
4. Choose a workflow that matches the task.

## The product shape changed

### Try this

- Use an image that shows the complete product outline.
- Avoid extreme camera angles unless they are intentional.
- Ask Hollapic to preserve the exact proportions and packaging shape.
- Use a reference scene with a similar product angle.

## The label, logo, or text changed

### Try this

- Upload a larger, sharper product image.
- State that the label layout and logo must remain unchanged.
- Avoid tiny source labels.
- Provide exact text for new labels, banners, or infographics.
- Review all AI-generated text before publishing.

## Product Swap replaced the wrong object

### Try this

- Upload your product first and the reference scene second.
- Say which object should be replaced.
- Ask Hollapic to keep all unrelated scene objects unchanged.
- Use a reference scene with one clear main product.

## Style Copy followed the reference too closely

### Try this

- Ask for “lighting and mood only.”
- Request a new environment and composition.
- Avoid references dominated by another brand’s logo or packaging.
- Use the reference for visual direction, not exact copying.

## The wrong person appeared

### Try this

- Describe the role, pose, crop, clothing style, and interaction.
- Do not ask Hollapic to recreate a specific real person.
- Ask for a new person with a similar role rather than the same identity.

## Generation failed

### Try this

1. Check that the selected workspace has enough credits.
2. Confirm all images required by the workflow are uploaded.
3. Try a smaller PNG or JPG.
4. Refresh the page and check history.
5. Try again with a shorter instruction.

## Download failed

### Try this

- Confirm the generation completed.
- Refresh and open the result again.
- Download from Generation History.
- Try another browser if the button still does not respond.
