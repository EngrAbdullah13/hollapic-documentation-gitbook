# Label changed

Use this guide when a product label changes unintentionally.

## Fixes

1. Use a source image with a readable label.
2. Ask for the label layout and placement to remain unchanged.
3. Avoid asking for new label text unless you want Label Generation.

## Important behavior

For non-label workflows, label preservation is a product identity rule.
