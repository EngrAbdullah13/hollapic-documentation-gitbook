# Wrong human model generated

Human appearance can vary when a workflow uses hands, people, or lifestyle scenes.

## Fixes

1. Describe the role, crop, pose, and interaction.
2. Do not ask for a specific real person from a reference image.
3. Use Style Copy only for broad human scene direction.
