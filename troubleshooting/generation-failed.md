# Image generation failed

Generation can fail during validation, credit deduction, prompt planning, image generation, output storage, or job processing.

## Check first

1. Confirm the active workspace has enough available credits.
2. Confirm required workflow images are uploaded.
3. Try a smaller or clearer source image if upload failed.
4. Refresh and check whether the job appears in history.

## Important behavior

The backend stores failed job status and an error message when available.
