# Product was replaced incorrectly

This usually happens when the selected workflow does not match the intended task.

## Fixes

1. Use Background Replace if the product should remain and only the environment changes.
2. Use Product Swap if your uploaded product should be inserted into a reference scene.
3. Upload product first and reference second.
