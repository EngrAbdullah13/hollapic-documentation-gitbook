# Workspace invitation problems

Use this guide when an invite cannot be created or accepted.

## Common causes

- The collaborator already has active access.
- A pending invitation already exists for the same email.
- The invitation expired.
- The collaborator signed in with the wrong email address.
- A limited allocation exceeds free/shared credits.
