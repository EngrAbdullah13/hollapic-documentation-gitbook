# Workspace switching problems

Use this page when a shared workspace does not appear or does not load.

## Fixes

1. Refresh the app.
2. Confirm the invitation was accepted.
3. Ask the owner to verify access has not been removed.
4. Switch back to Personal Workspace and try again.

## Important behavior

If active membership is missing or revoked, the backend resets the user to their own workspace.
