# Reference style was copied too literally

Reference images should guide style, not duplicate another brand’s exact artwork.

## Fixes

1. Use instructions such as “use the lighting and mood only.”
2. Avoid reference images dominated by another brand’s logo or packaging.
3. Ask for a new composition inspired by the reference rather than a duplicate.
