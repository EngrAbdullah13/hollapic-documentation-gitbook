# Logo or text is incorrect

AI-generated text can be unreliable unless the source text is clear or the user provides exact words.

## Fixes

1. Use a product image where logo and label text are readable.
2. Provide exact visible words only when needed.
3. Avoid asking for inferred text.
4. Regenerate with clearer preservation instructions.
