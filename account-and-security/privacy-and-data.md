# Privacy and data

Hollapic stores account, generation, payment, Brand DNA, workspace, and asset metadata required to run the product.

## Public policy

The public privacy policy is available at:

```text
/privacy-policy
```

## Important behavior

Brand DNA summaries and profile data are user-scoped. Shared workspace data is resolved through owner workspace authorization.
