# Sign up

Hollapic signup uses email OTP verification.

## Steps

1. Open the signup form.
2. Enter email and profile details.
3. Request an OTP code.
4. Enter the six-digit code.
5. Continue into the app.

## Expected result

The backend creates the user if the email does not already exist and grants signup credits when eligible.
