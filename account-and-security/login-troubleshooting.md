# Login troubleshooting

Use this page when login or signup does not complete.

## Code is rejected

Request a new code and use the latest email. Older unused codes are invalidated when a new code is requested.

## No email received

Check spam and verify the email address. If repeated sends are blocked, see [OTP limit exceeded](../troubleshooting/otp-limit-exceeded.md).
