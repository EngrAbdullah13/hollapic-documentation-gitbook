# Sign Up, Log In, and Email Codes

Hollapic uses a six-digit code sent to your email for signup and login.

## Sign up

1. Open the signup page.
2. Enter your email and profile information.
3. Request the email code.
4. Open the latest email from Hollapic.
5. Enter the six-digit code.
6. Continue into the app.

## Log in

1. Open the login page.
2. Enter your email.
3. Request a code.
4. Enter the newest six-digit code.

Older codes may stop working after you request a new one, so always use the latest email.

## Code did not arrive

* Check the email address for typing mistakes.
* Look in Spam, Junk, Promotions, or Updates.
* Wait briefly before requesting another code.
* Use only the latest code you receive.

## Send limit

A single email address can request up to four successfully sent codes within one hour. A fifth request is blocked until the remaining wait time has passed.

The message should tell you approximately how many minutes remain.

## Code is rejected

1. Request one new code.
2. Open the newest Hollapic email.
3. Enter the code exactly as shown.
4. Make sure you are using the same email address on the login page.
